Abstract

概要

RFC 5328 registered the Uniform Resource Name (URN) namespace "dvb" for the Digital Video Broadcasting Project. This document updates RFC 5328 with new registrant information.

RFC 5328は、デジタルビデオブロードキャストプロジェクトのUniform Resource Name（URN）名前空間「dvb」を登録しました。このドキュメントは、RFC 5328を新しい登録者情報で更新します。

1. Introduction
1. はじめに

[RFC5328] registered the Uniform Resource Name (URN) namespace "dvb" for the Digital Video Broadcasting Project. Since the publication of that RFC, the registrant contact information has changed; this document updates the registrant information to make it current. It uses a role-based email address to reduce the need for updates in the future.

[RFC5328]は、デジタルビデオブロードキャストプロジェクトのUniform Resource Name（URN）名前空間「dvb」を登録しました。そのRFCの公開以降、登録者の連絡先情報は変更されました。このドキュメントは、登録者情報を更新して最新のものにします。役割ベースのメールアドレスを使用して、将来の更新の必要性を減らします。
