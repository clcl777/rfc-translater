Revision to Registration Procedures for Multiple BMP Registries
複数のBMPレジストリの登録手順の改訂
Abstract
概要

This document updates RFC 7854, "BGP Monitoring Protocol (BMP)", by changing the registration procedures for several registries. Specifically, any BMP registry with a range of 32768-65530 designated "Specification Required" has that range redesignated as "First Come First Served".

このドキュメントは、いくつかの登録手順を変更することにより、RFC 7854「BGP監視プロトコル（BMP）」を更新します。具体的には、「必要な仕様」と指定された32768-65530の範囲のBMPレジストリは、その範囲が「First Come First Sever」と再指定されています。

1. Introduction
1. はじめに

[RFC7854] creates a number of IANA registries that include a range of 32768-65530 designated "Specification Required". Each such registry also has a large range designated "Standards Action". Subsequent experience has shown two things. First, there is less difference between these two policies in practice than there is in theory (consider that [RFC8126] explains that for Specification Required, "Publication of an RFC is an ideal means of achieving this requirement"). Second, it's desirable to have a very low bar to registration, to avoid the risk of conflicts introduced by use of unregistered code points (so-called "code point squatting").

[RFC7854]は、「必要な仕様」と指定された32768-65530の範囲を含む多くのIANAレジストリを作成します。このようなレジストリには、「標準アクション」と呼ばれる範囲が広いです。その後の経験により、2つのことが示されています。第一に、実際にはこれら2つのポリシーの間で理論的には違いが少なくなります（[RFC8126]は、「RFCの公開はこの要件を達成する理想的な手段である」と説明していると説明しています）。第二に、未登録のコードポイント（いわゆる「コードポイントスクワッティング」）を使用することで導入される競合のリスクを回避するために、登録に対して非常に低いバーを持つことが望ましい。

Accordingly, this document revises the registration procedures, as given in Section 2.

したがって、このドキュメントは、セクション2に示されているように、登録手順を修正します。

2. IANA Considerations
2. IANAの考慮事項

IANA has revised the following registries within the BMP group:

IANAは、BMPグループ内で次のレジストリを修正しました。

* BMP Statistics Types

* BMP統計タイプ

* BMP Initiation and Peer Up Information TLVs

* BMPの開始と情報TLVを覗き込みます

* BMP Termination Message TLVs

* BMP終端メッセージTLV

* BMP Termination Message Reason Codes

* BMP終了メッセージ理由コード

* BMP Route Mirroring TLVs

* TLVをミラーリングするBMPルート

* BMP Route Mirroring Information Codes

* BMPルートミラーリング情報コード

For each of these registries, the ranges 32768-65530 whose registration procedures were "Specification Required" are revised to have the registration procedures "First Come First Served".

これらの各レジストリについて、登録手順が「仕様が必要」である範囲32768-65530は、登録手順を「最初に提供する」ために修正されます。

3. Security Considerations
3. セキュリティに関する考慮事項

This revision to registration procedures does not change the underlying security issues inherent in [RFC7854].

登録手順のこの改訂は、[RFC7854]に固有の根本的なセキュリティ問題を変更しません。
