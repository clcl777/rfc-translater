Revision to Registration Procedures for IS-IS Neighbor Link-Attribute Bit Values
IS-ISネイバーリンクアトリブビット値の登録手順の改訂
Abstract
概要

RFC 5029, "Definition of an IS-IS Link Attribute Sub-TLV", defines an IANA registry called "IS-IS Neighbor Link-Attribute Bit Values". This document changes the registration procedure for that registry from "Standards Action" to "Expert Review". This document updates RFC 5029.

RFC 5029、「IS-ISリンク属性Sub-TLVの定義」は、「IS-IS Neiver Link-Attributeビット値」と呼ばれるIANAレジストリを定義します。このドキュメントは、そのレジストリの登録手順を「標準アクション」から「エキスパートレビュー」に変更します。このドキュメントは、RFC 5029を更新します。

1. Introduction
1. はじめに

[RFC5029] defines the "IS-IS Neighbor Link-Attribute Bit Values" registry and specifies a registration procedure of "Standards Action". In practice, this registration procedure is unnecessarily restrictive, as it prevents allocation of bits to experimental protocols, which in turn increases the risk of conflicts introduced by use of unregistered code points (so-called "code point squatting").

[RFC5029]は、「IS-IS NeighterIS Link-Attribute BIT値」レジストリを定義し、「標準アクション」の登録手順を指定します。実際には、この登録手順は、実験プロトコルへのビットの割り当てを防ぎ、未登録のコードポイント（いわゆる「コードポイントスクワッティング」）の使用によって導入される競合のリスクを高めるため、不必要に制限されます。

Accordingly, this document changes the registration procedure for the registry, as described in Section 2.

したがって、このドキュメントは、セクション2で説明されているように、レジストリの登録手順を変更します。

2. IANA Considerations
2. IANAの考慮事項

IANA has changed the registration procedure for the "IS-IS Neighbor Link-Attribute Bit Values" registry to "Expert Review" [RFC8126] and has listed this document as a reference for the registry. General guidance for the designated experts is defined in [RFC7370], and more specific guidance can be found in [RFC5029].

IANAは、「IS-IS Neighbor Link-Attribute Bit Vate」レジストリの登録手順を「Expert Review」[RFC8126]に変更し、このドキュメントをレジストリの参照としてリストしました。指定された専門家の一般的なガイダンスは[RFC7370]で定義されており、[RFC5029]にはより具体的なガイダンスがあります。
